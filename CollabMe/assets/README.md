# Assets

Drop your brand art here, then re-add the references in `app.json`:

- `icon.png` — 1024×1024 app icon
- `adaptive-icon.png` — 1024×1024 Android adaptive foreground
- `splash.png` — splash screen image
- `favicon.png` — web favicon

Until then, Expo falls back to its default placeholders so the app still runs.
